# Phase 0: main (`python_src`) vs `misc\depr` (`spm_dir_norm`, `spm_vec`, `spm_unvec`)

This note compares the **main** translations in `python_src/` with the **alternative** ones in `misc/depr/`, using MATLAB in `matlab_src/` via the MATLAB Engine (`eng`, `rgms` conda env) as oracle; only `spm_unvec` imports another of the three within each stack (`python_src.spm_vec` vs `misc.depr.spm_vec`). The eleven `tests\oracle` scenarios were also run against the **`misc\depr`** implementations in `misc\depr\test_cross_oracle_on_alternative_spm_*.py`, and the twenty-four expanded **`misc\depr`** scenarios were also run against **`python_src`** in `misc\depr\test_cross_misc_depr_on_canonical_spm_*.py`. A single pytest over the six primary test modules plus those six cross shims gave **63 passed, 7 failed**, all off-diagonal.
